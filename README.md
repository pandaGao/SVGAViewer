# SVGA Viewer

Preview .svga files online

[Online Demo](http://www.pandagao.com/SVGAViewer/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
